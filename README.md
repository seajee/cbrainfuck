# CBrainfuck

Yet another Brainfuck compiler.

## Build
To build the project use the following commands

```bash
mkdir build
make
```

## Usage

```
Usage: cbf -i <input file>

Options:
  -h         Show this message
  -i <file>  Input source file
  -o <file>  Output file
  -t         Transpile only to output file
```
